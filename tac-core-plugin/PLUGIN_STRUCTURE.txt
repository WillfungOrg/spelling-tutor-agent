================================================================================
QUICK START
================================================================================

1. Copy plugin to your project:
   cp -r tac-core-plugin /path/to/project/.claude/plugins/tac-core

2. Install dependencies:
   pip install anthropic python-dotenv pydantic requests

3. Set required environment variable:
   export ANTHROPIC_API_KEY="sk-ant-..."

4. Run /begin in Claude Code:
   - AI analyzes your project
   - Auto-generates project-context.md
   - Recommends optimal workflow

5. Start coding with AI assistance!
